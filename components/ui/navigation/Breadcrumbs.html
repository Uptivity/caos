Accessibility Features

Built-in Accessibility:

  • ARIA Labels: Each breadcrumb nav has proper aria-label
  • Semantic HTML: Uses nav, links, and proper structure
  • Keyboard Navigation: All links are keyboard accessible
  • Focus Indicators: Clear focus states for navigation
  • Screen Reader: Current page identified appropriately
  • Touch Targets: Minimum 32px height for mobile users

Usage Guidelines:

  • Always wrap breadcrumbs in a nav element with aria-label
  • Use links for navigable items, spans for current page
  • Include home icon for better visual hierarchy
  • Collapse on mobile for long paths (use "..." indicator)
